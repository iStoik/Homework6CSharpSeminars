Пример:
0, 7, 8, -2, -2 -> 2

#### Задача 43:

_Напишите программу, которая найдёт точку пересечения двух прямых, заданных уравнениями y = k1 * x + b1, y = k2 * x + b2; значения b1, k1, b2 и k2 задаются пользователем._

```text
Пример:
b1 = 2, k1 = 5, b2 = 4,
k2 = 9 -> (-0,5; 5,5)
```
